- "entity_type": one of the defined entity types

3. Capture temporality carefully: "history of X" = historical, "admitted for X" = current
4. Map to target items when confident; leave empty if unsure
5. Include abbreviated mentions (MI, CHF, COPD, HTN, etc.)
